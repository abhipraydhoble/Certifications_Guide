## 🧱 AWS Certification Levels

AWS certifications are divided into **four levels**:

1. **Foundational**
2. **Associate**
3. **Professional**
4. **Specialty**

Each level builds on the previous one.

---

## 1️⃣ Foundational Level

## AWS Cloud Practitioner (CLF-C02)

### 🎯 Focus

* Understand **what AWS services do**
* Learn **basic cloud concepts**
* Introduction to **security and billing**

### 🔹 Compute

* Amazon EC2
* AWS Lambda
* Amazon Lightsail

### 🔹 Storage

* Amazon S3
* Amazon EBS

### 🔹 Networking

* Amazon VPC
* Amazon Route 53

### 🔹 Databases

* Amazon RDS
* Amazon DynamoDB

### 🔹 Security & Governance

* AWS IAM
* AWS Trusted Advisor
* AWS Shield

### 🔹 Billing & Cost Management

* AWS Cost Explorer
* AWS Budgets

---

## 2️⃣ Associate Level

## AWS Solutions Architect – Associate (SAA-C03)

### 🎯 Focus

* **Designing AWS architectures**
* Understanding **how services connect**
* High availability, scalability, and fault tolerance

### 🔹 High Availability

* Elastic Load Balancer (ELB)
* AWS Auto Scaling

### 🔹 Decoupling & Messaging

* Amazon SQS (Queues)
* Amazon SNS (Notifications)
* Amazon EventBridge

### 🔹 Advanced Storage

* Amazon EFS
* AWS Storage Gateway
* Amazon FSx

### 🔹 Databases

* Amazon Aurora
* Amazon ElastiCache
* Amazon Redshift

### 🔹 Security

* AWS KMS (Encryption)
* AWS Secrets Manager

---

## 3️⃣ Professional Level

## AWS Solutions Architect – Professional (SAP-C02)

### 🎯 Focus

* **Complex, enterprise-scale architectures**
* Multi-account environments
* Automation, governance, and migrations

### 🔹 Hybrid & Networking

* AWS Direct Connect
* AWS Transit Gateway
* VPC Peering

### 🔹 Governance & Management

* AWS Organizations
* AWS Control Tower
* AWS Service Catalog

### 🔹 Migration Services

* AWS Application Migration Service (MGN)
* AWS Database Migration Service (DMS)

### 🔹 Data Analytics

* AWS Glue
* Amazon Kinesis
* Amazon Athena

### 🔹 Automation

* AWS CloudFormation
* StackSets (Multi-region & Multi-account)

---
## 4️⃣ Specialty Level

### AWS Specialty Certifications

**Focus:**

* Deep expertise in **specific domains**
* Advanced troubleshooting & design

### 🎯 Available Specialty Certifications

* **AWS Security – Specialty (SCS-C02)**
* **AWS Advanced Networking – Specialty (ANS-C01)**
* **AWS Machine Learning – Specialty (MLS-C01)**
* **AWS Data Analytics – Specialty (DAS-C01)**
---

| Certification Level | Base Fee | Total with 18% GST |
| ------------------- | -------- | ------------------ |
| Cloud Practitioner  | $100     | ~₹10,000           |
| Associate           | $150     | ~₹15,000           |
| Professional        | $300     | ~₹30,000           |
